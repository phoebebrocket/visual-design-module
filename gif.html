Animated GIF

Explaining how I did it:

  • I began by filling the logo svg I had made with colour, and imported it into Adobe After Effects.
  • Using transform, I set a rotation keyframe at 0:00f in the timeline.
  • I then rotated the image 180 degrees and set another keyframe at 01:00f.
  • I rotated the image another 180 degrees and set the final keyframe.
  • As I wanted the gif to continuously rotate in a smooth manner, I set the final keyframe at 02:01f.
  • By doing this, it prevented the image looping 360 degrees and stopping before starting again.
  • I then exported the gif by adding it to the Adobe Media Encoder Queue
  • I could then save the gif by selecting 'Animated GIF' in Media Encoder and saving it locally as a .gif file.
  • I would use this gif on social media as part of a campaign to capture the attention of users.

Explaining how I did it:

  • For fun, I also created a hand-drawn image in Procreate on an iPad.
  • I created a new artboard, and drew an image of a cookie.
  • Using animation asssist, I then erased parts of the cookie and added dots for 'crumbs' on subsequent artboards to make it look like it was being eaten.
  • I set it to run on a loop, with 3 frames per second, and removed the background to make it transparent.
  • I could then export it as an animated GIF and saved it as 'web-ready' to reduce the file size.
  • I could use this gif for part of a social media campaign to be displayed on Instagram or Facebook and give a friendly feel to the brand.
